### ✨ Java 개인 mini-project
캐릭터정보를 생성, 조회, 수정, 삭제해보자

#### 캐릭터 관리 프로그램 - user.dat 파일에 Character 데이터를 관리

##### 프로그램은 아래 5개의 기능을 가집니다.
> 1. 캐릭터 생성
> 2. 캐릭터 정보 조회
> 3. 캐릭터 정보 수정
> 4. 캐릭터 삭제
> 5. 프로그램 종료
